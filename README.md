# finsyt

Finsyt is an AI-powered financial intelligence platform.

## Quick start

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## API endpoints

- `GET /api/v1/status`
- `POST /api/v1/research`

## Environment

Copy `.env.example` to `.env` and fill provider keys.
